# rfinalproject
### For my final project I looked at a F1 Strategy dataset.
This dataset includes data each lap for each race from 2022 - 2025.
There was no null values but there was a "none" value in the compound column. I was able to find out what tire compound it was supposed to be and updated the data.
There was data from preseason tests but I only wanted the data from the races so I created a new dataframe that was only races.
The average lap time is 92.19148 seconds.
The quickest laptime was 67.012 done by Max Verstappen at the Austrian Grand Prix in 2022. I also noticed that all of the quickest lap times were done at that Grand Prix.
The Austrian Grand Prix's average lap time is almost 10 seconds quicker than the next Grand Prix's.
I then looked that the average lap time a quickest lap time for each driver. Nyck de Vries had the lowest average lap time but he only raced in 11 races which is about a half a year.
I then looked at the average lap time for each tire compund. The soft tire had the lowest average laptimes. I then created a new column to see the tire life.
I created a chart to show how much each tire compound was used.
